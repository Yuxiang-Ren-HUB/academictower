# 学术魔塔 Academic Tower

> 一座二十三层的塔，装着每一个研究生走过或未曾走完的路。

一款以研究生生涯为背景的 RPG 闯关游戏。你将选择一位导师，收集资源，击败每层的「怪物」，登顶魔塔，完成学业。

## 特色

- **4 条路线**：善良导师 / 科研大佬 / 人脉老师 / 邪恶导师，命运各不相同
- **23 层关卡**：文献迷宫、数据荒漠、审稿人诘问、答辩深渊……
- **NPC 对话**：每个角色都有自己的故事
- **成就系统**：隐藏结局等你解锁
- **跨平台**：支持 Windows / Android / Web

---

## 运行方式

### 方式一：直接运行 exe（Windows）

从 [Releases](../../releases) 下载 `学术魔塔.exe`，双击运行，无需安装任何环境。

### 方式二：源码运行

**环境要求**
- Python 3.10+
- pygame 2.x

```bash
pip install pygame
python main.py
```

## 操作说明

| 按键 | 功能 |
|------|------|
| 方向键 / WASD | 移动 |

| 数字键 1 | 确认选择 |
| 数字键 0 | 放弃（慎重）|
| Q | 使用技能 |
| ESC | 返回 / 跳过 |
